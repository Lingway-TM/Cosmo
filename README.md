### 🌌 О проекте
**The Space Bot** — это не просто Telegram-викторина. Это интерактивный «интерфейс присутствия», созданный для погружения в историю и тайны космонавтики. Проект объединяет образовательный контент и эстетику «замершего кадра» — визуальный стиль, вдохновленный архивными снимками и интерфейсами орбитальных станций.

### 🛠 Стек технологий
* **Python 3.x** — основная логика системы.
* **pyTelegramBotAPI** — взаимодействие с Telegram API.
* **LingwayTM Design Framework** — концепция визуального повествования (AI-generated visuals).

### 🛰 Ключевые особенности
* **20+ уникальных уровней:** От первых спутников до горизонтов событий.
* **Визуальное погружение:** Каждый вопрос сопровождается атмосферным медиа-контентом в стиле «Cinematic Sci-Fi».
* **Система званий:** Гибкая оценка результатов — от «Гражданского» до «Легенды Галактики».
* **Open Source:** Код полностью открыт для изучения и модификации.

### ⚙️ Установка и запуск
1. Клонируйте репозиторий:
   
   git clone [https://github.com/LingwayTM/Cosmo.git](https://github.com/LingwayTM/Cosmo.git)
Установите зависимости:

pip install pyTelegramBotAPI

Вставьте ваш TOKEN в файл Cosmo.py.

Запустите систему:

python Cosmo.py
